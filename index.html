Este aplicativo reúne os principais comandos do Executar do Windows para facilitar seu acesso e agilizar tarefas comuns no sistema. Abaixo estão os comandos mais usados com suas descrições.

Principais Comandos do Executar

  • devmgmt.msc - Gerenciador de Dispositivos
  • printmanagement.msc - Gerenciador de Impressoras
  • msconfig - Configurações do Sistema
  • msinfo32 - Informações do Sistema
  • regedit - Editor do Registro
  • gpedit.msc - Editor de Diretiva de Grupo
  • firewall.cpl - Firewall do Windows
  • services.msc - Serviços
  • ncpa.cpl - Conexões de Rede
  • control - Painel de Controle
  • sysdm.cpl - Propriedades do Sistema
  • desk.cpl - Propriedades de Vídeo
  • taskmgr - Gerenciador de Tarefas
  • cmd - Prompt de Comando
  • powershell - PowerShell
  • calc - Calculadora
  • notepad - Bloco de Notas
  • explorer - Explorador de Arquivos
  • cleanmgr - Limpeza de Disco
  • dfrgui - Desfragmentador
  • taskschd.msc - Agendador de Tarefas
  • perfmon - Monitor de Desempenho
  • resmon - Monitor de Recursos
  • UserAccountControlSettings - Controle de Conta de Usuário
  • mstsc - Conexão de Área de Trabalho Remota
