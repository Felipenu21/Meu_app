App - Comandos Executar do Windows

Aprenda e utilize rapidamente os principais comandos do Executar no Windows

O que você encontra no aplicativo

O aplicativo reúne e explica os comandos mais úteis da caixa "Executar" do Windows, ajudando técnicos e usuários a acessar ferramentas rapidamente.

Principais Comandos

  • cmd - Abre o Prompt de Comando
  • msconfig - Configuração do Sistema
  • services.msc - Gerenciador de Serviços
  • regedit - Editor de Registro
  • control - Painel de Controle
  • taskmgr - Gerenciador de Tarefas
  • appwiz.cpl - Programas e Recursos
